Partytown Example

This example page has included the Google Tag Manager (GTM) script.

If you view the source of this HTML page, you'll find the standard GTM script which the service tells you to copy and paste "as close to the opening <head> tag as possible".

The main difference however, is that this page's GTM script also has the type="text/partytown" attribute added to the opening <script> tag. Additionally, near the closing </body> tag is the Partytown library script.
